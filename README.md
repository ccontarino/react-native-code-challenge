# Expo Router Example

Use [`expo-router`](https://expo.github.io/router) to build native navigation using files in the `app/` directory.

## 🚀 How to use

```sh
# Install dependencies
npm install 
# Run expo local
npm run start
# Run ios
npm run ios
# Run android
npm run android

```
